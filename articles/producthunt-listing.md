**Description**:

Onicode is a free, open-source desktop app that combines AI chat with 80+ built-in tools, an autonomous browser agent, and multi-agent orchestration.

Three modes for everyone:
🗨️ Chat — General AI assistant
⚡ Workpal — Productivity partner
🛠 Projects — Software engineering

Highlights:
🌐 Browser Agent — AI controls Chrome: navigates, fills forms, extracts data
🤖 Multi-Agent — Parallel specialist sub-agents with own browser tabs
🔐 Credential Vault — AES-256 encrypted with OS Keychain
📅 Workflows — 7 step types + cron scheduler
🎨 12 Themes — Sand, Midnight, Obsidian, Ocean, and more
🔌 MCP — 55+ server catalog for Google Sheets, databases, Slack
💻 5 AI Providers — OpenAI GPT-5, Claude, Ollama (local)

MIT licensed. Free forever.

**Maker comment**:

Hi Product Hunt! I built Onicode because I was frustrated that every AI tool was just a chatbot. I wanted something that actually does things — edits files, runs commands, browses the web, manages projects.

The browser agent is the feature I'm most proud of. You tell the AI "go to Amazon and find the cheapest drone under $50" and watch Chrome open and do it automatically. Each sub-agent gets its own tab, so you can run multiple web tasks in parallel.

Would love your feedback. What would you add to an AI workspace?
