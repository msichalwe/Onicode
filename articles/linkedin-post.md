🚀 Excited to share what I've been building: Onicode

An open-source, AI-powered workspace that goes beyond chat. It has 80+ built-in tools, an autonomous browser agent, and multi-agent orchestration — all in one desktop app.

What makes it different:

🌐 Browser Agent — The AI opens a real Chrome window and autonomously navigates websites, fills forms, extracts data, and manages multiple tabs. You can literally watch it work.

🤖 Multi-Agent System — Spawn specialist sub-agents (researcher, implementer, tester, browser agent) that work in parallel, each with their own browser tab.

💼 Not Just for Developers — Three modes: Chat (general), Workpal (productivity), and Projects (coding). Built for writers, researchers, students, and professionals too.

🔐 Security First — AES-256 encrypted credential vault with OS Keychain integration. Your secrets never leave the main process.

Tech: Electron 34, React 19, TypeScript, SQLite, 5 AI providers (OpenAI GPT-5, Claude, Ollama local models).
