# Onicode — Product Vision

## What Is Onicode?

Onicode is an **AI-powered development environment** that flips the traditional IDE paradigm on its head.

Instead of "a code editor with AI bolted on" (like Cursor/Windsurf), Onicode is:

> **A premium AI chat that expands into a full VS Code IDE on demand.**

You open Onicode like opening ChatGPT — warm, inviting, conversational. When you need to write code, the IDE slides out seamlessly. When you're done, collapse back to chat.

## Core Principles

1. **Chat-first** — The default experience is conversational, not overwhelming
2. **IDE on demand** — Full VS Code power when you need it, hidden when you don't
3. **Connected** — GitHub, Gmail, and services integrated natively
4. **Portable** — Your API keys, preferences, and knowledge follow you across projects
5. **Multi-provider** — Bring your own AI: OpenAI, Anthropic, Ollama, or OniAI
6. **Premium feel** — Warm themes, smooth animations, luxury UX
7. **Mobile-ready** — Chat with your AI and run commands from your phone
